Overview

Use this guide to connect Sedifex to websites, automation services, and internal systems.

Typical integration flow

  1. Generate API credentials in your Sedifex account.
  2. Send authenticated requests to Sedifex endpoints.
  3. Receive webhook events in your backend.
  4. Store and sync booking/customer/payment updates in your own systems.

Best practices

  • Keep API keys in server-side secrets only.
  • Verify webhook signatures before processing.
  • Use idempotency keys and retry-safe handlers.
